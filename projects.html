Daily UI Challenge - UI/UX Project

Project Type: Current personal Project

Description:

A challenge where I design and create UI everyday for a hundred days! Even though it's a UI challenge I also take it as an opportunity to practice and learn more about UX design. Also, all the UI I have created so far have been shared on twitter!


Project Length:

A hundred days (or more)!


Role in the project:

Since it's a personal project I am responsible for all stages of design and development.


Tools:

For the daily UI challenge I use various tools depending on what I want to accomplish. The tools that are essential to this project are my notebooks, pens, pencils, and my graphics tablet. Aside from the essentials, I use Sketch, Principle, Mockplus, and MyPaint.

Travel Canada - Website and Database Project

Project Type: Academic Project

Description:

A website and database project created for my Database Management Systems course in university. My group and I were inspired by travel so we decided to create a project that would motivate people to travel to Canada. We wanted to accomplish this task by providing people with a website that provides information on restaurants, activities, accommodations, and shopping areas in different parts of Canada.


Project Length:

3 months


Role in the project:

Contributed to the development of the database, populating the database, web development, PHP scripting, and querying.


Markup and Programming Languages:

HTML, CSS, PHP, SQL, JSON


Tools:

mySQL Workbench, MAMP, NetBeans IDE, Google Drive

UniCook - Human Computer Interaction Project

Project Type: Academic Project

Description:

UniCook was a project for my Human Computer Interaction course in university. Over three months we were to create different prototypes while practicing the task-centered system design(TCSD) method. What inspired this project was the thought of providing our fellow university students with a desktop app that gives them recipes that they can cook given their time constraints and budget.


Project Length:

3 months


Role in the project:

Contributed to the design and development process by participating in brainstorming ideas, conducting user interviews, prototyping, and coding some of the functionalities.


Markup and Programming Languages:

XAML, Visual C#


Tools:

Visual Studio 2015, Microsoft Blend for Visual Studio, GitHub
